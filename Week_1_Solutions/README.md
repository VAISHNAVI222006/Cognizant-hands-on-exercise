
# Week 1: Engineering Concepts - Data Structures and Algorithms

##  Overview

Week 1 focuses on **core Data Structures and Algorithms** that form the foundation of software engineering. These concepts are essential for writing efficient, scalable, and maintainable code.

##  Topics Covered

1. **HashMap & O(1) Operations** - Inventory Management System
2. **Linear & Binary Search** - E-commerce Platform Search Function
3. **Bubble Sort & Quick Sort** - Sorting Customer Orders
4. **Array Operations** - Employee Management System
5. **Singly Linked List** - Task Management System
6. **Search Algorithms Comparison** - Library Management System
7. **Recursive Algorithms** - Financial Forecasting

##  Skills Learned

- Choosing appropriate data structures based on problem requirements
- Analyzing time and space complexity of algorithms
- Implementing searching and sorting algorithms
- Understanding recursion and its optimization
- Comparing algorithm performance
